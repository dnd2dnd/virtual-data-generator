# virtual-data-generator
## 개요
move-modeling에서 사용할 수 있는 가상의 데이터를 생성해주는 것 입니다.

## 환경
```
* Platform : Android Studio
```

## 사용 방법

1. 작업자, 이동 장비 둘 중의 하나 유형을 선택한다.

2. equ_no, Serail No를 입력한다
